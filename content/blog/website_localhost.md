在网页（index.html）所在目录
```bash
python3 -m http.server
```
会显示：
```bash
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```
此时，访问[http://localhost:8000/](http://localhost:8080/)即可看到网页。

退出：Ctrl+C
