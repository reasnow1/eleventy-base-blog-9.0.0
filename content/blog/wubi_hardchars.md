| 字 | 五笔编码 | 拆解提示 |
|----|----------|----------|
| 凹 | MMGD | 凹字形拆为“冂、冂、一、𠃊” |
| 凸 | HGMG | 凸字形拆为“丨、一、冂、一” |
| 盛 | DNNL | “成”取前三笔（D N N）+ “皿”（L） |
| 贼 | MADT | “贝”（M）+ “戎”（A D T） |
| 熬 | GQTO | “敖”（G Q T P）去末笔换“灬”（O） |
| 夕 | QTNY | 字根，在Q里，然后丿（T）乛（N）丶（Y） |
| 尴 | DNJL | “厂”（D）+ “乚”（N）+“刂”（J）+ “皿”（L） |
| 戈 | AGNT | 一（A），戈（G），乛（N），丿（T） |
| 拜 | RDFH | “手”（R）+ “三”（D）+ “十”（F）+ 末笔识别 |
| 帧 | MHHM | “巾”（M H）+ “贞”（H M ） |
| 丫 | UHK | “丷”（U）+ “丨”（H）+ 识别码 K |
